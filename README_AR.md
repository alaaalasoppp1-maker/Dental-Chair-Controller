# Dental Chain Chair Controller v1

برنامج Windows مستقل يعمل كوسيط بين:

- برنامج Dental Chain OS لاحقًا.
- مجلد صور SOPRO محليًا.
- تطبيق Chair Display APK عبر شبكة العيادة المحلية.

## الوظائف الحالية

- فهرسة جميع المجلدات الفرعية داخل SOPRO.
- مراقبة الصور الجديدة بدون عرض تلقائي.
- عرض أحدث صورة عند الطلب.
- السابق والتالي.
- تكبير وتصغير وتحريك وإعادة ضبط.
- `Ctrl + زر الـ backquote/ذ` لعرض أحدث صورة.
- الأسهم و`+/-` أثناء عرض الصورة.
- `Esc` لإغلاق الصورة.
- شاشة ترحيب / مريض / خدمات / أطفال.
- System Tray.
- حفظ الإعدادات.
- تشغيل اختياري مع Windows.
- خادم WebSocket محلي على المنفذ 8765.
- بناء Portable وInstaller عبر GitHub Actions.

## التشغيل محليًا

```powershell
npm.cmd install
npm.cmd start
```

## بناء EXE عبر GitHub

1. أنشئ Repository خاصًا جديدًا.
2. ارفع محتويات هذا المجلد كاملة، بما فيها `.github`.
3. افتح تبويب Actions.
4. شغّل `Build Chair Controller for Windows`.
5. نزّل Artifact باسم:
   `Dental-Chair-Controller-Windows`

داخله:
- Installer EXE.
- Portable EXE.

## عنوان الاتصال

يظهر داخل الواجهة مثل:

```text
ws://192.168.1.20:8765
```

يجب أن يتصل تطبيق APK بنفس العنوان مؤقتًا. سنضيف الاكتشاف والاقتران التلقائي في الإصدار التالي.

## ملاحظة الصور

الصور تُقرأ من جهاز العيادة وتنتقل ضمن الشبكة المحلية فقط. لا يتم رفعها إلى Firebase أو أي خدمة سحابية.
